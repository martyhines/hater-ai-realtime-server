# HOW TO SET UP PROMOTIONAL TEXT:

1. Go to App Store Connect
2. Select your app
3. Go to "App Store" → "iOS App" (or version)
4. Scroll to "Promotional Text" section
5. Enter your text (max 170 characters)
6. Save changes (no app submission required!)

# BEST PRACTICES:

✅ Keep under 170 characters
✅ Include emojis for visual appeal
✅ Add urgency or exclusivity
✅ Highlight unique features
✅ Include clear call-to-action
✅ Test on different devices
✅ Update regularly to keep fresh

# TIMING STRATEGIES:

• Launch week: Feature announcements
• Holidays: Seasonal promotions
• Slow periods: Special offers
• New features: Update announcements
• User feedback: Address common requests
